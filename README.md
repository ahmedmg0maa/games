# Family Detective Files

لعبة تحقيق عربية تفاعلية بواجهة RTL احترافية، جاهزة للرفع على GitHub Pages.

## التشغيل
افتح `index.html` مباشرة في المتصفح، أو ارفع المجلد كاملًا على GitHub Pages.

## المحتوى
- 3 قضايا: متوسطة، صعبة، شديدة الصعوبة
- مشتبه بهم متعددون ودوافع حقيقية
- مستندات: شرطة، طب شرعي، كاميرات، رسائل، اتصالات، بريد
- لوحة أدلة تفاعلية
- دفتر ملاحظات وحفظ محلي LocalStorage
- صفحة تلميحات تدريجية منفصلة
- تقرير نهائي وتحليل للحل

## بدون صوتيات
تم تنفيذ المشروع بدون أي ملفات صوتية بناءً على الطلب.
